# Datasets

Esta pasta contém os datasets utilizados nos exercícios e projetos do curso.

## Organização

Os datasets são organizados por módulo ou projeto quando possível.

## Fontes Comuns de Dados

- [Kaggle](https://www.kaggle.com/datasets)
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)
- [Google Dataset Search](https://datasetsearch.research.google.com/)
- [data.gov](https://www.data.gov/)
- [Dados Abertos Brasil](https://dados.gov.br/)

## Formato dos Arquivos

- CSV (Comma-Separated Values)
- Excel (.xlsx, .xls)
- JSON
- Parquet
- SQL databases

## Nota

Datasets grandes não devem ser commitados no repositório. Use o arquivo `.gitignore` para excluí-los e adicione instruções no notebook sobre como baixá-los.

Para datasets grandes, considere:
- Adicionar um script de download
- Usar links diretos para download
- Documentar a fonte original
