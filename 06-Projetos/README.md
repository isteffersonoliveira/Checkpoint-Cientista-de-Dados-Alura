# Projetos

Este diretório contém projetos completos de Ciência de Dados que integram os conceitos aprendidos.

## Tipos de Projetos

- Análise exploratória de dados
- Modelos de predição
- Sistemas de recomendação
- Análise de séries temporais
- Classificação de imagens
- Processamento de linguagem natural

## Estrutura de Projeto

Cada projeto deve conter:
- README.md com descrição do projeto
- Notebooks com análise e desenvolvimento
- Scripts Python quando aplicável
- Dados (quando possível) ou instruções para obtê-los
- Resultados e visualizações

## Notebooks

Os notebooks de projetos estão na pasta `notebooks/`.
