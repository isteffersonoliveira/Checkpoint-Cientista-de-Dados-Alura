# Visualização de Dados

Este módulo cobre técnicas e ferramentas para visualização de dados.

## Conteúdo

- Matplotlib básico
- Seaborn para visualizações estatísticas
- Plotly para gráficos interativos
- Tipos de gráficos:
  - Gráficos de linha
  - Gráficos de barras
  - Histogramas
  - Box plots
  - Scatter plots
  - Mapas de calor
- Boas práticas de visualização
- Dashboards

## Notebooks

Os notebooks de exercícios e exemplos estão na pasta `notebooks/`.
