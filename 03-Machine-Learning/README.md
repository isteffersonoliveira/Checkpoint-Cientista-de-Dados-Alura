# Machine Learning

Este módulo cobre os principais algoritmos e técnicas de Machine Learning.

## Conteúdo

- Aprendizado supervisionado
  - Regressão Linear
  - Regressão Logística
  - Árvores de Decisão
  - Random Forest
  - SVM
  - K-Nearest Neighbors
- Aprendizado não supervisionado
  - K-Means
  - Clustering Hierárquico
  - PCA
- Validação de modelos
- Feature Engineering
- Métricas de avaliação

Os notebooks de exercícios e exemplos estão na pasta `notebooks/`.
