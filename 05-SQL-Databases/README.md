# SQL e Bancos de Dados

Este módulo cobre SQL e manipulação de bancos de dados para Ciência de Dados.

- Fundamentos de SQL
- Consultas básicas (SELECT, WHERE, ORDER BY)
- Joins (INNER, LEFT, RIGHT, FULL)
- Agregações (GROUP BY, HAVING)
- Subconsultas
- SQLAlchemy para Python
- Integração com Pandas
- NoSQL básico
